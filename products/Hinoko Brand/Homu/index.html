Flagship Autonomous Agent

HOMU

最新鋭の国産モデルを搭載した、
次世代の自律型AIエージェント。

Concept

プライバシーと安全性を重視した、
信頼できるAI体験を提供する。

Homuは、日本国内サーバーで稼働する最新鋭の国産AIモデル（PLaMo / LLM-JP等）を搭載しています。

自律的に思考し、タスクを遂行する次世代のパートナーとして、あなたの日常と業務を強力にサポートします。

01

Domestic Model

PLaMoやLLM-JPなど、日本語の文脈を深く理解する最高峰の国産モデルを採用。自然で高度な推論を実現します。

02

Domestic Server

すべての処理は日本国内のサーバーで完結。機密データが海外に流出するリスクを根本から排除し、確固たるプライバシー保護を約束します。

03

Autonomous Action

指示を待つだけでなく、目的達成のために自ら計画を立て、ツールを操作し、タスクを自動で遂行します。

In Development
